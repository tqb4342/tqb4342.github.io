hibernate视频教程

  • Hibernate是一个开放源代码的对象关系映射框架，它对JDBC进行了非常轻量级的对象封装，它将POJO与数据库表建立映射关系，是一个全自动的orm框架，hibernate可以自动生成SQL语句，自动执行，使得Java程序员可以随心所欲的使用对象编程思维来操纵数据库。本教程主要包括：Hibernate框架概述&安装、 hibernatetools 插件、hibernate.cfg.xml 、hibernate 映射文件、详解 hibernate 映射文件等。
